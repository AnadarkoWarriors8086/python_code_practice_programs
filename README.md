# python_code_practice_programs
These were some programs I created for my Python Academy courses.

They are basic examples, but I am proud to say that I did not simply do what the others taking the course did. 
I would come up with an idea of how I can accomplish the task, and then I would set out to do it. 
My ideas were not always the best method, however, that is part of the learning process. Do it the hard way
and then learn the easier ways.

Dashboard.py is a personal expense tracking dashboard I am developing. It is in it's basic form, but will soon
evolve into a useful tool.
